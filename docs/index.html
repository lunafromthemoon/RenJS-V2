  • A Visual Novel creator made for writers!

  • Designed for Writers

    Create your games by writing a script akin to a screenplay, without the need to do any programming.

  • Easy to Customise

    The RenJS GUI Builder tool allows you to visually create impressive and unique GUIs, in a few simple steps.

  • Easy to Extend

    Add new game mechanics like hidden object, RPG features, and much more, with the Plugin System.

  • Multiplatform

    Export and package your games as Desktop and Mobile apps, with all the power from Electron and Cordova!

  • Easy to Pack and Ship

    You can package and distribute your games as Linux and Windows applications in a few simple steps with the RenJS Desktop Packager.

  • Phaser

    RenJS is powered by PhaserJS, a powerful 2D video game engine for HTML5. Thanks to it, a whole suite of tools to create special effects and most complex behaviours is readily available.

If this is your first time here and want to see what RenJS is all about, The Tutorial game will show you all about it. Play it to have a first look at how a RenJS game looks like, and learn the basics of the script language at the same time.
